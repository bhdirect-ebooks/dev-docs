## Classes

**For use with `<dl>`**
* `.abbr-list`

**For use with `<ol>`**
* `.digit`
* `.upper-roman`
* `.lower-roman`
* `.upper-alpha`
* `.lower-alpha`
* `.lower-greek`

**For use with `<ul>`**
* `.disc`
* `.circle`
* `.square`
* `.none`
